# 機能要件

### 入力項目

ユーザーが以下の条件を入力できるようにする：

### 開発言語

このプロジェクトの公用語は \*\*日本語\*\* です。
IssueやPRのやり取りは全て日本語で行ってください。
 
### コミットメッセージの規約

コミットメッセージは日本語で、以下のプレフィックスを推奨します。

`feat:` 新機能の追加
`fix:` バグ修正
`docs:` ドキュメントのみの変更
`style:` コードの動作に影響しない変更（空白、フォーマットなど）
`refactor:` リファクタリング（機能追加やバグ修正を含まない変更）

### 環境設定

文字コード:UTF-8 (BOMなし)
改行コード:LF

### 出力

# システム要件

* WEBブラウザ上で動作すること
* Dockerで動く開発環境生成、ローカルで動作可能にする
* 使用技術：バックエンド：Python（Flask）, フロントエンド：HTML / JavaScript
* 実行時ログは詳細に記録されること（デバッグや保守を考慮）
